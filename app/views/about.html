About My-Board!

This web application was created for the ChartJS Dashboard Challenge, and is open source.

View Project on Github
Overview

The purpose of this simple web application was to demonstrate using ChartJS charts to display visual data to the user in an engaging way. Instead of just using 3 charts to display data I wanted to take a step further and display different types of data using the ChartJS library.

Data Sources

The data comes from a variety of places, the most important things in someones life is there career, health and network, so the career data comes from my own personal bank account, career, the health data comes from my entries, and the network data comes from my social networks.
