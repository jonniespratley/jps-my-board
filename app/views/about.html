Overview

This application was created for the ChartJS Dashboard Challenge, the purpose was to demonstrate using ChartJS charts to display visual data to the user in an engaging way.

Data Sources

The data comes from a variety of places, the most important things in someones life is there career, health and network, so the career data comes from my own personal bank account, career, the health data comes from my entries, and the network data comes from my social networks.
